Features Command
Get installation candidate apt policy package
Get list of installed packages apt list --installed
Get list of installed packages including name apt list --installed | grep name
Get signature for apt-offline on offline machine sudo apt-offline set package.sig --install-packages package
Download zip of apt packages from signature file apt-offline get pacakge.sig --bundle package.zip
Install zip of apt packages from zip sudo apt-offline install package.zip
Get dependencies of a package apt depends package
Download dependencies recursively apt-get download $(apt-rdepends package | grep -v "^ " ) apt-cache depends --recurse --no-recommends --no-suggests --no-conflicts --no-breaks --no-replaces --no-enhances --no-pre-depends package | grep "^\w"
